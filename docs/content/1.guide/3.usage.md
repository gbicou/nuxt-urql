title: Usage
description: How to use the module
---

This module reexports @urql/vue composables :

- `useQuery`
- `useMutation`
- `useSubscription`
- `useClientHandle`
- `gql`
- `useUrqlClient`

Read [@urql/vue documentation](https://formidable.com/open-source/urql/docs/basics/vue/) to learn more.

---

## useQuery

```vue
<template>
  <div>{{ data }}</div>
</template>

<script setup lang="ts">
const { data } = useQuery({
  query: gql`
    query name {
      # ...
    }
  `,
});
</script>
```

## useQuery with SSR

Await query to run it on the server :

```vue
<template>
  <div>{{ data }}</div>
</template>

<script setup lang="ts">
const { data } = await useQuery({
  query: gql`
    query name {
      # ...
    }
  `,
});
</script>
```

## useUrqlClient

Composable to access the URQL Client instance

```vue
<template>
  <div>{{ data }}</div>
</template>

<script setup lang="ts">
const client = useUrqlClient();

const query = gql`
  query name {
    # ...
  }
`;
const { data } = await client.query(query, {}).toPromise();
</script>
```
